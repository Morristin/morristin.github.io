# 关于「蓝莓色硅谷」

> 那天晚上结束时，
>
> 我决定用最长的方式过马路。
>
> ——王家卫《蓝莓之夜》

## 关于蓝莓色

「蓝莓色」这个意向取自[潮汐](https://tide.fm/)声音场景「[蓝莓之夜](https://s.moreless.io/s/UNY7pxEA5I)」。该旋律的命名取自王家卫同名电影。

我从未看过王家卫的任何一部作品。对这位著名导演的印象也仅仅是他擅长于使用色调进行叙事。但我有印象，他的这个特点很突出，突出到仅仅是凭借一张画面便能让人猜出这是他的作品。

我有时会梦想我也能够如此。在我的心目中，无论是文字的排版、字体的选择，乃至于为文字服务的其他艺术形式的混合，都只不过是使得文字营造的意境更加清晰可感。人类使用语言进行思考与对话，而语言所能转化到视觉而不失去其含义的唯一途径即是文字。

我希望我所写的文字能够为你带来这样的体验。

潮汐所做的「蓝莓之夜」描绘了一处萨克斯悠扬的灯红酒绿之境。透过被雨水模糊的玻璃，世间的一切都显得那样轻浮，大大小小的光晕被抹去含义。我想，如果我在车内，可能面前会是一块微微发光的荧幕，上面的语句与外面的喧嚣一样泛着不同的颜色。
